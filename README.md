# FuelTankRefiller
A KSP Mod that adds a button to the right click menu of every part. When pressed, it refills the entire tank with all its fuel.

## How to Install
1. Go to the [Release](https://github.com/fmooij/FuelTankRefiller/releases/tag/0.0.1) and download the source code file (.zip).
2. Merge the GameData folder inside of the .zip file with your KSP GameData directory. (most likely located at C:\Program Files (x86)\Steam\steamapps\common\Kerbal Space Program\GameData\)

You should now when you start a flight, be able to refill your fuel tanks.
